# 📊 Refinery Performance & Predictive Analytics

### End-to-End Data Analytics Project using Python, Machine Learning & Power BI

---

## 📖 Project Overview

This project presents an end-to-end refinery analytics solution to evaluate operational performance, analyze profitability, forecast future business performance, and visualize insights through interactive Power BI dashboards.

The project combines Exploratory Data Analysis (EDA), Machine Learning, Time Series Forecasting, and Business Intelligence to identify key revenue drivers, predict future Revenue and Gross Refining Margin (GRM), and provide actionable business insights.

---

## 🎯 Project Objectives

- Analyze refinery operational and financial performance.
- Identify key factors affecting Revenue and Gross Refining Margin (GRM).
- Forecast future Revenue and GRM using Machine Learning.
- Perform scenario analysis to evaluate the impact of market and operational changes.
- Develop an executive Power BI dashboard for business performance monitoring.
