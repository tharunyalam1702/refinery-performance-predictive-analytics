# 📊 Refinery Performance & Predictive Analytics

### End-to-End Data Analytics Project using Python, Machine Learning & Power BI

---

## 📖 Project Overview

This project presents an end-to-end refinery analytics solution to evaluate operational performance, analyze profitability, forecast future Revenue and Gross Refining Margin (GRM), and visualize business insights through interactive Power BI dashboards.

The project integrates Exploratory Data Analysis (EDA), Machine Learning, Time Series Forecasting, and Business Intelligence to support data-driven decision-making.

---

## 🎯 Project Objectives

* Analyze refinery operational and financial performance.
* Identify key drivers affecting Revenue and Gross Refining Margin (GRM).
* Forecast future Revenue and GRM using Machine Learning.
* Perform scenario analysis to evaluate the impact of business and market variables.
* Build an interactive Power BI dashboard for performance monitoring and business insights.
